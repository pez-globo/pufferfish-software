# Services( PCAC )

Scenario: The Active service parameter in services class is not equal to HFNC or PCAC
Given: A Services class object is constructed with default all_states parameter
When: parameter service value in parameter class of all_states is not equal to pc_ac or HFNC
Then: Alarm Limits are not updated as per Alarm Limit requests

Scenario: The Active service parameter in services class is equal to PCAC
Given: A Services class object is constructed with default all_states parameter
When: parameter service value in parameter class of all_states is equal to pc_ac
Then: Alarm Limits are updated for PCAC service class

Scenario: The Alarm limits are updated for SpO2 when Active service is equal to PCAC
Given: A Services class object is constructed with Active service equal to PCAC
When: The requested lower value for the SpO2 alarm limits range is lesser than the maximum allowed lower value for the range.
Then: Alarm Limits are not updated as per Alarm Limit requests

Scenario: The Alarm limits are updated for SpO2 when Active service is equal to PCAC
Given: A Services class object is constructed with Active service equal to PCAC
When: The requested upper value for the SpO2 alarm limits range is greater than the maximum allowed upper value for the range.
Then: Alarm Limits are not updated as per Alarm Limit requests

Scenario: The Alarm limits are updated for fio2 when Active service is equal to PCAC
