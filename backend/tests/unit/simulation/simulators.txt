# Services (PCAC)

Scenario: The Service class correctly handles PCAC simulator
Given: A Service class object is constructed with default parameters
When: The ventilation mode in parameters class is not equal to PCAC
Then: The simulator parameter values are unchanged

Scenario: The Service class correctly handles PCAC simulator
Given: A Service class object is constructed with default parameters
When: The ventilating value in parameter class is false
Then: The simulator parameter values are unchanged

Scenario: The Service class correctly handles PCAC simulator
Given: A Service class object is constructed with default parameters
When: The current time and previous time are same
Then: The sensor measurments value for flow is unchanged

Scenario: The Service class correctly handles PCAC simulator
Given: A Service class object is constructed with default parameters
When: ventilator is in inspiratory phase
Then: The Sensor measurment values are updated in inspiratory phase

Scenario: The Service class correctly handles PCAC simulator
Given: A Service class object is constructed with default parameters
When: ventilator is in expiratory phase
Then: The Sensor measurment values are updated in expiratory phase
