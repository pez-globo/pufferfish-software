# Service (HFNC)

Scenario: The parameters Service class correctly updates HFNC parameters
Given: A Service class constructed with default parameters
When: parameter service value in parameter class of all_states is not equal to HFNC or PCAC
Then: parameters are not updated as per the parameter requests

Scenario: The parameters Service class correctly updates HFNC parameters
Given: A HFNC Service class constructed with default parameters
When: parameter service value in parameter class of all_states is equal to HFNC
Then: The value for ventilating in parameters is updated as per parameter request

Scenario: The parameters Service class correctly updates HFNC parameters
Given: A HFNC Service class constructed with default parameters
When: The current parameter value for fio2 is greater than fio2_min and the requested value for fio2 is less than fio2_min
Then: The parameter value is updated to fio2_min

Scenario: The parameters Service class correctly updates HFNC parameters
Given: A HFNC Service class constructed with default parameters
When: The current parameter value for fio2 is less than fio2_max and the requested value for fio2 is greater than fio2_max
Then: The parameter value is updated to fio2_max

Scenario: The parameters Service class correctly updates HFNC parameters
Given: A HFNC Service class constructed with default parameters
When: The parameter request value for flow is greater than 0 and less than 80
Then: The parameter value for flow is updated with requested value

Scenario: The parameters Service class correctly updates HFNC parameters
Given: A HFNC Service class constructed with default parameters
When: The parameter request value for flow is less than 0 or greater than 80
Then: The parameter value for flow is Unchanged
