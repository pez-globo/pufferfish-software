ventserver package¶

Modules and scripts for the Pufferfish ventilator’s backend server.

Sub-packages:

util: utility modules, classes, and functions used by other modules. sansio: abstract interfaces and basic reusable components for writing

protocols in a sans-I/O discipline.

protocols: Sans-I/O protocols implementing all protocol logic for the

Pufferfish ventilator’s backend server.
